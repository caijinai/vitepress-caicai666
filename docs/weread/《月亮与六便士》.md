# 《月亮与六便士》

《月亮与六便士》是威廉·萨默赛特·毛姆创作的一部经典小说，讲述了一个证券经纪人追求艺术理想的故事，探讨了理想与现实之间的冲突。

### 小说背景

《月亮与六便士》首次出版于1919年，灵感来源于法国后印象派画家保罗·高更的真实经历。小说的主人公查尔斯·斯特里克兰德原本是一名成功的证券经纪人，拥有美满的家庭和稳定的职业，但他在中年时突然抛弃一切，前往巴黎追求绘画的梦想。

### 情节概述

斯特里克兰德的行为让周围的人难以理解，他在异国他乡经历了贫困和饥饿的折磨，同时也在精神上承受着巨大的痛苦。最终，他逃离文明世界，来到南太平洋的塔希提岛，与当地土著人共同生活，找到了创作的灵感和内心的宁静。尽管他在艺术上取得了巨大的成就，但在临终前，他却要求将自己的作品焚毁，反映了他对艺术与世俗生活的复杂态度。


### 主题分析

小说的标题“月亮”象征着崇高的理想和艺术追求，而“六便士”则代表着世俗的物质生活。毛姆通过斯特里克兰德的故事探讨了理想与现实之间的矛盾，反映了20世纪初期西方社会对物质文明与精神追求的思考。

  
这部作品通过细腻的叙述和深刻的主题，成为了探讨人性与社会矛盾的重要文学作品。


### 原文摘抄

“世界上只有少数人能够最终达到自己的理想。我们的生活很单纯、很简朴。我们并不野心勃勃，如果说我们也有骄傲的话，那是因为在想到通过双手获得的劳动成果时的骄傲。我们对别人既不嫉妒，更不怀恨。”——《月亮和六便士》·五十四


我们每个人生在世界上都是孤独的。每个人都被囚禁在一座铁塔里，只能靠一些符号同别人传达自己的思想；而这些符号并没有共同的价值，因此它们的意义是模糊的、不确定的。我们非常可怜地想把自己心中的财富传送给别人，但是他们却没有接受这些财富的能力。因此我们只能孤独地行走，尽管身体互相依傍却并不在一起，即不了解别人也不能为别人所了解。——引自第190页


这一定是世间无数对夫妻的故事。这种生活模式给人以安详亲切之感。它使人想到一条平静的小河，蜿蜒流过绿茸茸的牧场，与郁郁的树荫交相掩映，直到最后泻入烟波浩渺的大海中。但是大海却总是那么平静，总是沉默无言、声色不动，你会突然感到一种莫名的不安。也许这只是我自己的一种怪想法（就是在那些日子这种想法也常在我心头作祟），我总觉得大多数人这样度过一生好象欠缺一点什么。我承认这种生活的社会价值，我也看到了它的井然有序的幸福，但是我的血液里却有一种强烈的愿望，渴望一种更狂放不羁的旅途。这种安详宁静的快乐好象有一种叫我惊惧不安的东西。我的心渴望一种更加惊险的生活。只要在我的生活中能有变迁——变迁和无法预见的刺激，我是准备踏上怪石嶙峋的山崖，奔赴暗礁满布的海滩的。
