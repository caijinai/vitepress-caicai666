# 12月23日 
**恢复训练！好累啊！**


- 上上周四开了楼下健身房的卡，虽然设备都不算新，场地环境也一般。但是聊胜于无吧。健身房还是得选方便的！
- 给自己定的计划是隔天练一次，一三五六锻炼，其他时间休息。不想跟以前一样每天都练，那样身体恢复不过来，整个人都好累。也很庆幸最近工作不算忙，可以按计划去锻炼。

- 恢复锻炼，发现力量真的下降好多好多！o(╥﹏╥)o
  - 从上上周五启动锻炼，第一节送的私教课带练了臀和背，练完以后大腿和屁股酸痛了4天，走路都不利索…🤣
  - 以前臀腿硬拉至少都能拉100斤，最高能拉140斤了。昨天晚上拉空杆（20kg）都累，最后也只勉强加了10kg负重…
  - 还有同样的训练计划以前练下来最多觉得稍微累，现在已经减少重量和数量，还是觉得好喘好晕…
  - 以前练臀腿感觉力量感满满，现在练着就容易腰酸，核心力量也降好多了。
- 感觉到了巨大的退步真是会有种灰心的感觉呀！练完就憧憬着什么时候才能恢复以前的状态！😔
- 只能慢慢加油了，急也急不来。要做一个有正常期待的成年人，总不能停训一年多还以为体能不会下降吧，总不能以为随便练几天就能恢复吧。运动变强是长久坚持的事情ヾ(◍°∇°◍)ﾉﾞ
