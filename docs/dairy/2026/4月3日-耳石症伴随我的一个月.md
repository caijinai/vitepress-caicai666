# 4月3日-耳石症伴随我的一个月

3月2日，本来是很平常的一天。早起准备去上班的，结果忽然一阵晕眩。糟糕，耳石症又发作了。我想着让自己安静一下，度过起床之后的那段难受期就好了吧，还是可以坚持去上班，毕竟上班也不需要频繁地弯腰和走动。结果在刷牙，换衣服之后还是不行，晕眩地已经没法站着了。赶紧请假，然后喊朗哥起床送我去看医生。

到了医院，跟上次一样跟医生简单做了沟通，接着就去护士那做复位和艾灸。护士做了几次，眼睛没有眼颤了。虽然还是有点晕，但医护人员说是正常情况，需要慢慢恢复。我就回家了，下午正常上班。

到晚上下班回家也一切都正常，感觉自己也不晕了。觉得复位成功，好了。emm然后还不知死活地熬夜到12点多才睡觉。结果第二天起床，还是又微微的晕眩感。有点难受，但是还能坚持换衣服啥的去上班。就是在地铁上没座位，只能靠着地铁里的扶手杆有个支撑感觉好点。整个头都是胀胀的，昏昏的。好难受（那刻真觉得牛马好可怜，生病了也不舍得请假，能扛就扛…）。当天就下班回家处理完各种事情以后，十点多就睡，果然下一天起床就感觉好多了。虽然并没有完全好。

我想着坚持不请假吧，就这样先熬着，反正也是低头的时候才会感觉晕，不影响上班就好。一直坚持到了周六去看医生

医生觉得之前人工复过一次了还没完全好，那就上机器吧。开单以后，我就打车去北区的医院看。第一次见到耳石症复位机器啊，不知道的以为这是个模拟太空舱呢😐医生让我坐到设备自带的凳子上，给我五花大绑以后，给我带上了头盔和眼部摄像头（观察我的眼颤情况）。机器复位感觉就是比人工的摇摆角度更大一些。我看不到自己处于什么视角，我只是感觉到自己好像被倒立过。复位以后，医生观察我的眼睛没有眼颤的情况了，就说好了。但是我心里挺忐忑地，因为上次人工复位也说好了，回去还是不行。很想跟医生说再复一次再确认吧，然而就是没说出口啊！嗐！

医生过来给我解开机器以后就嘱咐了一些注意事项，可我还是觉得晕晕的，问是不是正常情况，医生说是，不会立刻好。好吧，于是我在医院的凳子上坐了会就打车回家了。可是回家一直还是晕晕的啊……很不安心。到了晚上，我想说试试，于是低下头看看。结果看到了熟悉的旋转画面…😔😔今天可是花了600大洋看医生啊，都上机器了还没复位成功！

立刻挂了周日的号，到了医院以后医生就说再人工复一下。可是护士姐姐复了好多次都没见好…真是令人沮丧啊😭医生就说那就只能回去再休养休养，气血虚补补身体，过几天再来复位看看。唉，好吧。还能怎么样呢。

然后我就开始了与耳石症“如履薄冰”地相处的一个月。咱就是说，没啥年假的牛马人就不请假再去复位了，每次去都得去掉一个上午。而且症状也不严重了，只要不低头太久，问题都不大。耳石症在这一个月里也变成我有没有好好休息的晴雨表了。如果我熬夜，第二天肯定会晕的厉害点。如果一直早睡，睡够了，第二天就没啥感觉，而且晕眩转动的角度也越来越小。我也听医生的话，稍微补一下身体。很认真地吃祛湿茶，吃一些温补的东西。也认真地吃鱼油、钙片和维D了

只不过耳石症确实也是对日常生活有影响的，因为频繁地走动或弯腰低头就会觉得整个人好像在晕船一样，所以这一个月也不咋运动，能坐着就坐着。家里乱糟糟地也不想收拾了…😂每天帮朗个做个鸡胸肉带饭已经是很辛苦了。

我查资料说，耳石症在一个月内自愈的概率是50%。我还很消极的想这50%怕是归不上我身上。结果4月2号那天，锁单车的时候忽然觉得不晕了。平时锁单车都感觉要头胀个几秒钟。回家试了一下躺着向左转头，也没有出现晕眩的画面了！低头也不晕了！噢耶！我的生活又恢复这个昵称了。

总结下来，因为耳石症跑了3次医院，花了将近1000蚊😭，最后还是靠自愈的…即使是小病小痛也是很影响心情和生活状态的。身体健康真的太重要了，以后我给别人写祝福我就都写“身体健康”。以后我也要坚持早睡，多休养，积极进补。希望耳石症不要再找上我了。
