# 菜菜版焖面

两人份：菜菜和朗朗（非减脂期时），总热量大概3500大卡
  

# 食材准备

1、五花肉：300g，切小片备用

2、长豆角：300g，切段（一定得是长豆角，扁豆、四季豆都没有长豆角焖出来的味道好吃）

3、黄豆芽：300g，洗干净

4、鲜面条：500-600g

4、料头：生姜两三片，大蒜两三颗拍碎，小辣椒一两颗切粒

5、香料：花椒一小把、八角1个、干辣椒3个（要用辣的那种）

6、半个大蒜（量多），剁成蒜蓉或用搅拌机搅碎

7、芝麻香油

  

  

# 烹饪：

1、热锅加少量油，先把五花肉煸炒，煸出油，表面有点焦就行。（看出油情况，如果出油太多就捞出一些油）

2、加入姜片煎香，再放入大蒜和香料，小火炒香

3、五花肉和香料们炒匀，沿锅边淋入一圈料酒、一圈半酱油、一点老抽上色、一勺蚝油，继续翻炒出酱香

4、加入长豆角翻炒一两分钟，加入开水没过锅中食材，水可以稍多一丢丢。加两小勺盐（3-4g），煮两三分钟

5、加入黄豆芽继续煮，将豆芽稍微煮软就好。

6、这个时候可以看看水量有没有没过所有食材，高出1cm左右，水少可以补一点。试试汤的味道，不够咸可以加点盐。

7、捞出小半碗汤汁

8、将面条均匀码在食材上，撒松一点，让面条之间有比较多空隙。然后将小半碗汤汁淋在面上

7、盖盖子，**焖7分钟**。看看汤汁是不是差不多收干，不能收太干，还需要留一点汤汁把面条拌匀。

8、倒入蒜末，芝麻香油，将面条和其他食材拌匀，整锅端上桌

香喷喷！
