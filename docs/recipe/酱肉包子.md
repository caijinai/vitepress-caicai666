# 酱肉包子

**备料：**

* **包子面团：**

1. 500g中筋面粉
    
2. 5h酵母
    
3. 1勺猪油
    
4. 270g温水
    

```
💡tips：可放冰箱隔夜醒发
```



* **内馅：**

1. 五花肉末，肥瘦3:7先煸油，再放瘦肉末炒匀（约1斤肉？）
    
2. 姜末、生抽、老抽、蚝油、甜面酱、十三香
    
3. 炒香
    
4. 再加入葱花（加一丢丢丢小苏打防止变色）、洋葱碎、胡萝卜碎、拌匀
    

  

**操作：**

1. 面团醒发以后，搓成长条
    
2. 切小面团，50g一个
    
3. 包馅，总共10个包子。
    
4. 包子放温水隔着醒发20分钟
    
5. 醒发以后，开火水开转小火蒸12分钟，焖3分钟
