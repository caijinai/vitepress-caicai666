
# 冯小厨版卤牛肉

4斤肉的量

## 前提准备：


炒糖色：小半勺食用油、加入一勺白糖、一勺清水，糖：水=1:1，小火慢熬，一个方向搅拌，大气泡变成小气泡，炒到枣红色再加入开水，就可以。

1、牛肉放清水里浸泡3小时去除血水，最好每个小时换一次水，更有效去血水
2、焯水：捞出后加水烧开，不需要加去腥三件套（姜、葱、料酒）煮热以后如有浮沫就撇掉，水烧开以后煮5分钟左右
3、捞出以后，可以炸一些小孔，好入味
4、料头：生姜切片，红葱头（或洋葱、大葱）切块、带根香菜
5、香料：八角、香叶、桂皮、干辣椒、陈皮、白芷、白蔻、小茴香、去籽草果、花椒（emm，用卤料包代替吧）
6、将料头炒香，先炒料头，再炒香料，全部炒香以后加入100g料酒，170g酱油，中火煮一分钟左右，激发酱油的香味。加入清水，再加入三勺提前炒好的糖色，三勺食盐，放入牛腱子。烧开后转最小火炖1.5小时
7、捞出料头和香料，牛肉继续放汤里泡着入味。
