# 木耳五花肉拌饭

**大胃两人份备菜**

1. 干木耳一把，提前浸泡，切丝
    
2. 香菇四五颗，提前浸泡
    
3. 青椒2颗，红椒1颗，切丝
    
4. 料头：①大蒜（量多）拍破；②指天椒1颗，切粒；③生姜、葱白，切粒
    
5. 五花肉半斤左右，也可加点瘦肉。切丝，白肉可以单独放
    
6. 瘦肉有需要可以先腌制（老抽、胡椒粉、半勺盐、淀粉、油）
    
7. 鸡蛋3~4个
    

**操作**

1. 煎蛋，油要多。煎到焦香以后铲小块，不需要铲得太碎
    
2. 五花肉肥的部分先炒出油
    
3. 加瘦肉翻炒
    
4. 加料头（葱姜蒜）
    
5. 淋一圈米酒或料酒去腥增香
    
6. 倒入木耳
    
7. 调味：盐、糖、鸡精or味精、一圈酱油沿锅边淋入、小半勺老抽上色
    
8. 大火炒一分钟
    
9. 放青红椒、鸡蛋。翻炒均匀
    
10. 把菜铺平整，盖上半锅米饭压一下。（两杯米煮一锅，取半锅米饭或多一点也行）
    
11. 盖上锅盖，小火焖两分钟。开盖拌匀即可
