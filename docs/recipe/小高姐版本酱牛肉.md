# 小高姐酱牛肉

食材：牛肉 1.4KG 

1、泡水去除血水 

2、牛腱子肉放入270g酱油中，隔夜浸泡（第一酱） 

3、第二天，牛肉连酱油一起倒入锅中，加入冷水，水没过肉以后再多倒一点，煮热以后如有浮沫就撇掉

4、放大料：花椒、八角、桂皮、草果、香叶、陈皮（可选）、山楂（可选）、干辣椒（可选）

5、放酱：甜面酱160g（第二酱），带甜味，小麦淀粉和豆类淀粉长时间发酵的味道

6、放豆腐乳，25g（第三酱） 

7、汤煮开以后尝尝味道，适当补充盐。接着小火炖2小时12分钟，保持似开微开的状态，以免肉被炖散了。

8、关火，肉在锅里闷着，放凉。在冰箱里冰一个晚上再吃。
