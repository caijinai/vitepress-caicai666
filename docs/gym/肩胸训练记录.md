# 肩胸训练集合
## 胸肩（陆）
### 热身
- 招财猫
- 绕肩膀
- 原地小跑热身

### 正式训练
1.  **仰卧史密斯推胸** 
 - 重量：5组*12，加20kg
 - 要点：手要握宽，小手臂垂直地面
2. **斜卧哑铃推胸**
  - 重量：5组*12, 7.5kg
3.  **固定推胸**
  - 要点：先推直，收肩胛骨，腰微微镂空。注意不要耸肩，手肘尽量压平。向前推的时候要注意肩胛骨稳定，不要让肩胛骨往前送）
4.  **蝴蝶机夹胸**
   - 重量： 5组*12 12KG
   - 要点：始终挺胸，胸先发力，手臂放松，往前推直
5. **蝴蝶机推肩**
   - 重量：5组*12 12kg
   - 要点：面向座椅，手臂始终打直
6. **侧平举**
   - 重量： 5组*20 5-10kg
   - 要点：小臂放松，大臂用力上抬）
