# 训练记录
## 臀腿训练231109（黄）

### 拉伸
泡沫轴滚臀，大腿内侧
往前跨步压髂腰肌
90-90坐姿，往一侧拉伸
四足支撑，抬高屁股手摸对侧脚，拉伸大腿

## 激活
1-弹力带固定到小腿与膝盖中间，螃蟹步来回，中间深蹲开合跳20个
2-手扶凳子，撅屁股，稍微屈膝，大腿往侧边做外展，每边25个

## 训练
1-臀推器械
加片每边10kg，15个，2组
每边20kg，15个，3组。（怪累的）

2-哈克深蹲（注意膝盖不要内扣）
加片每边10kg，15个，1组
加片每边28kg？12个，4组

3-相扑硬拉
注意两脚打宽一点，膝盖向两边撑开，起来的时候膝盖不要完全打直的情况下屁股收紧
先小重量热身
杠20kg+片20kg，15个，1组
加片25kg，12个，4组
（左侧腿比较紧，下蹲的时候感觉拉得疼，影响训练。以后要多拉伸多放松左侧）

4-髋外展
50kg，坐姿两腿打开到最大支撑15秒，再在这个基础上往外做20个外展，尽量每个做完稍微停顿，最后一个停顿3秒

再换个小重量重复以上动作

一共4组

# 背部训练20231123（陆）

## 热身
1. 招财猫
2. 手臂展开转圈前后各10圈

## 正式训练：

1. 龙门架宽距高位下拉两组递减（8片~6片），12个，5组

2. 直臂下压，3片铁，15个，4组
   含胸，手要放松，用手掌的后半部分下压，手肘窝相对

3. 龙门架坐姿含胸划船，12个
   用窄距握把，不要挺胸，保持含胸往身体方向拉，重点锻炼下背

4. 颈后下拉，用中距握把，12个
   手的力线保持垂直，向下拉，两个肩胛骨夹紧

5. 杠铃划船➕半程硬拉，5组，12个
   不要通过晃动身体来拉，尽量感受是背部夹紧以后，往上拉杠铃

6. 卷腹➕二头弯举

    榨干最后一点力气……
