# 功能训练集合

## 20230912功能训练（黄）

### 热身

- 抱膝走，抬腿90°向一边打开
- 直腿跳着走回来
- 抬手单腿平衡走
- 高抬腿走
- 高抬腿侧着走
- 拉伸大腿后侧
- 拉伸臀部

### 正式训练

>  4个动作按顺序完成为一组，大概耗时4分钟。一共4组。一组完成后才可以休息，休息时间1-2分钟

#### 第一个组合

- 杠铃划船15次，20kg~25kg
- 波速球站立甩战绳30秒
- 毛毛虫爬行5个（膝盖不可以弯，手要多往前）
- 跳箱12个

#### 第二个组合

- 传统硬拉15个，40kg

- 撑箱子~俯卧撑（手宽距，胸要碰到箱子边缘）15个

- 仰卧抱球抬腿两头起，20个

- 深蹲开合跳20个

  > 💡：前面热身20分钟，加上两套动作下来正好一节课。练完好累，心率飙升150
