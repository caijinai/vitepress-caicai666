# 臀腿训练集合
## 臀腿训练231216（黄）
### 拉伸与热身
- 燕式平衡
- 后踢腿
- 拉伸大腿内侧（坐着双腿展开八字，身体往前趴）
- 青蛙趴
### 激活
- 平板支撑+手往前伸
- 无负重臀推
### 正式训练
- 负重臀推15+壶铃摇摆30,4组
- 哈克深蹲，4组*15，负重左右20KG
- 哈克机单腿深蹲，4组*12，注意动作要领是撅屁股，少屈膝。
- 直腿硬拉，5组*10-12个，逐渐加重。最后加到60KG
- 髋外展递减组（120磅-90磅），两个重量各20个为一组。4组

## 臀腿训练231109（黄教练）

### 拉伸
- 泡沫轴滚臀，大腿内侧
- 往前跨步压髂腰肌
- 90-90坐姿，往一侧拉伸
- 四足支撑，抬高屁股手摸对侧脚，拉伸大腿

### 激活
- 弹力带固定到小腿与膝盖中间，螃蟹步来回，中间深蹲开合跳20个
- 手扶凳子，撅屁股，稍微屈膝，大腿往侧边做外展，每边25个

### 正式训练
1. 臀推器械
- 加片每边10kg，15个，2组
- 每边20kg，15个，3组。（怪累的）

2. 哈克深蹲（注意膝盖不要内扣）
- 加片每边10kg，15个，1组
- 加片每边28kg？12个，4组

3. 相扑硬拉
- 注意两脚打宽一点，膝盖向两边撑开，起来的时候膝盖不要完全打直的情况下屁股收紧，先小重量热身。
- 空杠20kg+片20kg，15个，1组。加片25kg，12个，4组
（左侧腿比较紧，下蹲的时候感觉拉得疼，影响训练。以后要多拉伸多放松左侧）

3. 髋外展
- 50kg，坐姿两腿打开到最大支撑15秒，再在这个基础上往外做20个外展，尽量每个做完稍微停顿，最后一个停顿3秒
- 再换个小重量重复以上动作，一共4组

>💡：黄教练训练特点就是特别注重训练前的拉伸和激活，这两项基本会用掉半节课。但是整体感受很好，拉升跟激活到位了，训练起来动作都能做到位，训练状态也更好 <br>
⭐: 这一套训练下来，第二天屁股很酸

## 臀腿20231012（黄）
### 拉伸
- 泡沫轴滚臀，大小腿，重点滚大腿内侧
- 青蛙趴
- 拉伸大腿前侧
- 核心激活，四足支撑抬腿抬手支撑，10个
- 死虫式，仰卧，对侧手脚打开

### 热身
- 臀桥：弹力圈定位到膝盖，臀顶上来先撑住固定10秒，再动态做5个，再固定撑住5秒。（很酸爽
- 螃蟹步➕深蹲：换个弹力小一点的，固定到膝盖。螃蟹步走1个私教区的长度，再深蹲20个。重复两组。深蹲不用蹲太低，大腿跟地面平行就好

### 正式训练
1. **髋外展递减**
  - 打开以后要稍微停顿
  - 每一组重量都比前面的重，第每个重量做20个，一组重量：93-66，最后一组加到112-93
2. **相扑式深蹲**
  - 双脚距离尽量大，臀部往外撑，起来的时候不要打直膝盖，感觉大腿外旋让屁股收紧，膝盖尽量撑开
  - 重量：22.5kg，12个，4组
3. **传统硬拉**
  - 重量：空杆，40kg，50kg，50kg，每组12个
  - 臀部尽量向后撑开，脚距大概跟胯同宽，膝盖感觉要往外撑开（不要内扣）。杆只需要下到膝盖下方一点距离就行，不用下太多。（练完这个心率到150…要命）
4. **负重单腿蹲**
  - 重量：15kg，18个每边，4组
  - 手拉哑铃，一只脚支撑，一只脚往前站。蹲下，大腿跟地面平行。

>💡练后总结：累到不行。。很酸爽！
