# 臀腿训练记录
## 臀腿训练231109（黄教练）

### 拉伸
泡沫轴滚臀，大腿内侧
往前跨步压髂腰肌
90-90坐姿，往一侧拉伸
四足支撑，抬高屁股手摸对侧脚，拉伸大腿

### 激活
1-弹力带固定到小腿与膝盖中间，螃蟹步来回，中间深蹲开合跳20个
2-手扶凳子，撅屁股，稍微屈膝，大腿往侧边做外展，每边25个

### 正式训练
1-臀推器械
加片每边10kg，15个，2组
每边20kg，15个，3组。（怪累的）

2-哈克深蹲（注意膝盖不要内扣）
加片每边10kg，15个，1组
加片每边28kg？12个，4组

3-相扑硬拉
注意两脚打宽一点，膝盖向两边撑开，起来的时候膝盖不要完全打直的情况下屁股收紧
先小重量热身
杠20kg+片20kg，15个，1组
加片25kg，12个，4组
（左侧腿比较紧，下蹲的时候感觉拉得疼，影响训练。以后要多拉伸多放松左侧）

4-髋外展
50kg，坐姿两腿打开到最大支撑15秒，再在这个基础上往外做20个外展，尽量每个做完稍微停顿，最后一个停顿3秒

再换个小重量重复以上动作，一共4组

>💡：黄教练训练特点就是特别注重训练前的拉伸和激活，这两项基本会用掉半节课。但是整体感受很好，拉升跟激活到位了，训练起来动作都能做到位，训练状态也更好 <br>
⭐: 这一套训练下来，第二天屁股很酸
