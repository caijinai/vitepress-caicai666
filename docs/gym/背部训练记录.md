# 背部训练集合
## 背部训练20231123（陆）

### 热身
1. 招财猫
2. 手臂展开转圈前后各10圈

### 正式训练：

1. 龙门架宽距高位下拉两组递减（8片~6片），12个，5组

2. 直臂下压，3片铁，15个，4组
   含胸，手要放松，用手掌的后半部分下压，手肘窝相对

3. 龙门架坐姿含胸划船，12个
   用窄距握把，不要挺胸，保持含胸往身体方向拉，重点锻炼下背

4. 颈后下拉，用中距握把，12个
   手的力线保持垂直，向下拉，两个肩胛骨夹紧

5. 杠铃划船➕半程硬拉，5组，12个
   不要通过晃动身体来拉，尽量感受是背部夹紧以后，往上拉杠铃

6. 卷腹➕二头弯举

    榨干最后一点力气……

>💡：陆教练训练特点是一定要榨干你所有的力气和能量。不停地push你挑战高难度和高配重！比如刚拉完背看我状态还可以，就拿个哑铃让我继续做单边拉背…恶魔。但是有这样的刺激这样身体进步很明显hhh <br>

##  背20231016（黄）

### 热身

1. 燕式平衡
2. 高抬腿走
3. 向前跨步下压腿手举过头顶，伸展身体
4. 腿分开两倍肩宽，手摸地往前爬，一边脚往前踩，跟手掌位置持平。转身举手。（像最伟大拉伸）。再手摸地爬回来，向左摸脚尖，再向右摸脚尖
5. 核心激活，仰卧死虫式对侧手脚打开收回10次。再一起打开收回5次

### 正式训练

1. 杠铃划船（第1组：30kg15个；第234组：35kg12个）

2. 引体向上+燕式平衡（12个，4组）

3. 固定器械拉背+山羊挺身（臀部发力）

   双手拉27.5kg~30kg？，15个，4组

   单手拉32.5kg？，15个，4组。加山羊挺身（单手拉，左边比较稳定，发力感好）

4. 固定器械拉背（27.5kg，12个，4组）

5. 坐姿俯身拉背（锻炼下背部），15个，4片铁？

   用龙门架，坐在条凳上，屁股往后做，让身体和大腿的夹角小一点，固定好角度，挺直腰背，手向下拉，停顿一秒。
