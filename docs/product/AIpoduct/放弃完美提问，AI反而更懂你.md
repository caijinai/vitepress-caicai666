

# 放弃完美提问，AI反而更懂你

有没有这种时刻——你对着AI问了半天，越问越烦，最后只想关掉它。 你开始怀疑：是不是我不会写提示词？是不是别人用AI更聪明？后来我发现，问题根本不在你。 

大多数人用AI，像在问客服：想清楚问题→打字→等回复→用完就走。 可问题是——你真正卡住的时候，往往根本想不清楚该问什么。

你连自己在纠结什么都说不明白，却要AI给你一个好答案，这本身就很反人性。

后来我真的换了一种用法，不再逼自己把问题想清楚，而是直接用语音，把脑子里的想法说出来。

我用的是一个叫GetSeed的AI录音卡。它不会急着给你结论，而是先把你说的内容拆开、归类、找重复点，再把那些你反复提到、却没意识到的重要线索拎出来。

有时候它整理出来的，不是一个答案，而是几条可以继续往下想的方向，让你突然意识到：原来我真正卡住的是这里。那一刻我才知道，我不是不思考，只是终于不用一个人把所有想法都整理一遍了。

现在我给自己的规则很简单：

第一，不要再追求完美提问。说人话、说废话、说没想清楚的话，都行。

第二，能用语音就别打字。嘴比脑子快，很多真实想法只有说出来才会出现。

第三，让AI多提醒你，反问你，而不是只回答你。有时候被问一句：“你最近是不是一直在纠结这个？”比给你十个答案都管用。

慢慢你会发现，AI不再只是一个工具，而是在变成你思考过程的一部分。

从录音卡，到AI眼镜，当AI不再只存在于屏幕里，而是随时记录、提醒、补充你的思考，它正在变成一种真正意义上的“认知延伸”。 



> 转载自罗辑思维公众号，虽然是一篇引流贴，但是这些观点让我觉得如释重负。我一直觉得我如果要从AI那里获取到有用的东西，我必须写好一套清晰完整的提示词，因此当AI没有输出让我满意的东西的时候，我会怀疑是自己提示词写得有问题。“把AI当成思考过程的一部分”这个观点就很有意思，不一定要借助AI得到答案，通过AI帮助自己梳理思路也是一种使用方式。感觉有点像“橡皮鸭”原理呢。
